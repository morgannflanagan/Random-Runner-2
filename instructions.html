Instructions

All the information needed to play our game


Object of the game

The object of the game is to have Kiki the Platypus jump over bad objects, like the poisonous mushrooms. To earn points, Kiki will have to run and avoid those objects. To earn extra points, Kiki will also have to collect special items. Each different object will give them a different amount of points.

How to play

Use the right arrow key to help Kiki run and move forward and the up arrow to jump. Make sure to avoid any and all bad objects that might come your way.


Good objects

  • Berries

    • Berries will give you an extra five points.

  • Squirrels

    • Squirrels will give you an extra ten points.

    Objects to avoid

  • Poisonous mushrooms

    • Running into these items will cause Kiki to die and the game will start over.
